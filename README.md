# STM32-Secure-Boot


### Static Analysis & Coding Standards

- MISRA-C Standard: MISRA-C:2012 (with Amendments)
- MISRA Tool: Cppcheck MISRA addon
- Compiler: GCC with -Wall -Wextra -Wconversion enabled
- Scope: Application-level source code
- Vendor libraries (HAL/CMSIS) excluded from MISRA analysis
- Deviations documented in MISRA_Deviations.md
